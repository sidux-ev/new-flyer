Semplice-Linux project

Not the usual Debian remix, since 2009.

Debian is awesome: Everybody can modify, improve and remix it to fit their needs. It is a
great basis for a fast, lightweight, clean and simple distribution. This is the aim of
Semplice-Linux, which is build on Debian sid to be always on the edge.

Designed to be lightning fast on modern 64-bit machines, Semplice-Linux also works well on old glories, requiring only 384 MB of RAM. To provide a fast environment the distribution is based on Openbox and a set of lightweight applications.

Semplice-Linux is built using only official Debian staff, with a framework to ensure the same working environment across all builds. It is compiled according to the Debian Policy and every modification to the base system is available in our repositories. It is possible to get the same Semplice-Linux system by simply starting from a basic Debian bootstrap.

Semplice-Linux includes non-free firmware to
support a wide variety of machines, but makes it easy to remove them to encourage using a real open-source system. See
http://semplice-linux.org/doku/semplice:nonfree#installed_system for more information.

The base working system provides everything a common user needs:
* A word processor (Abiword)
* A spreadsheet (Gnumeric)
* A music player (Exaile)
* A web browser (Chromium)

It also includes the Synaptic Package Manager which lets you install the 38,000 packages that Debian offers!
